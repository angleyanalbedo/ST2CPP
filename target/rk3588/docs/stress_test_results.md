# RK3588 PLC 周期压力测试报告

Date: 2026-07-01
Hardware: MatriBox (RK3588)
Kernel: 5.10.199-xenomai-3.2.4-hdu PREEMPT IRQPIPE
CPU: Cortex-A76 @ cpu6 (isolcpus), SCHED_FIFO/90, mlockall
Cycle: 1000us (timerfd), 15s per run

## 负载说明

stress_test_pou.cpp 模拟真实工业控制场景的复合负载：

- 级联 PID 回路（浮点密集）
- FIR 滤波器（数组/乘加密集）
- 齐次坐标矩阵变换（4×4 / 8×8 浮点密集）
- 复杂状态机（逻辑分支密集）
- 位打包/解包（位操作密集）
- IEC 定时器/计数器链（功能块密集）
- 大数组滑动平均（内存带宽密集）
- ProcessImage I/O 读写

## 结论

1. **Load=5** 是 RK3588 (A76@cpu6) 在 1ms 周期下的推荐安全负载上限
2. **Load=8** 需 2ms 周期或优化矩阵运算（改用 Neon/SIMD）
3. **Load=10** 超出现有周期预算，不建议 1ms 下使用
4. 调度器本身零超限、零错误、零看门狗触发 — 运行时调度器稳定性验证通过
5. 抖动主要来自 POU 执行时间的周期波动，而非调度器或 timer 抖动
